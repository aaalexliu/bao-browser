# Bao - landing page mockups

Static HTML concepts for the Bao marketing site. Each file is fully self-contained
(inline CSS + inline SVG mascot, no assets, no build). Open any file directly in a
browser.

Audience lead: **prosumer / government-forms wedge**. Primary CTA: **Add to Chrome**.

## Landing directions

| File | Direction | Feel |
|---|---|---|
| [`1-cute-playful.html`](1-cute-playful.html) | Cute & playful | Warm pastels, floating bun mascot, sparkles, rounded cards |
| [`2-clean-saas.html`](2-clean-saas.html) | Clean modern SaaS | Whitespace, faux product screenshot, metric row, feature grid |
| [`3-bold-techy.html`](3-bold-techy.html) | Bold & techy | Dark mode, monospace, workflow-IR terminal, "the LLM is not the runtime" |
| [`4-editorial-story.html`](4-editorial-story.html) | Editorial story | Serif headlines, paper texture, numbered chapters, pull quote |

## Icon

The mascot is a hand-drawn inline SVG with two forms, defined once per page:

- `#bao` - **squircle badge** (orange gradient tile). The brand mark: nav, footer, app/store icon.
- `#baobun` - **bare bun** (no ground). The mascot: big hero moments.

Chosen design: squircle badge, content `^ ^` eyes, no cheeks, three small steam wisps,
roomy even padding.

- [`icons/icon-final.html`](icons/icon-final.html) - the chosen icon at 16/32/48/128 + placements
- [`icons/icon-explorations.html`](icons/icon-explorations.html) - the original 12-direction exploration
